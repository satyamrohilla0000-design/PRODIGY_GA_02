# Image Generation with Pre-trained Models

## Overview
This project focuses on generating images from text prompts using pre-trained generative AI models such as DALL-E Mini and Stable Diffusion.

## Objectives
- Understand text-to-image generation.
- Explore diffusion-based generative models.
- Generate realistic images from textual descriptions.
- Compare outputs from different pre-trained models.

## Technologies Used
- Python
- Stable Diffusion
- DALL-E Mini
- PyTorch
- Diffusers Library

## Workflow
1. User provides a text prompt.
2. Prompt is processed by the pre-trained model.
3. Latent image representation is generated.
4. Diffusion process creates the final image.
5. Output image is displayed and saved.

## Features
- Text-to-image generation
- High-quality image synthesis
- Customizable prompts
- Support for multiple generation models

## Sample Prompts
- "A futuristic city at sunset"
- "A robot painting a landscape"
- "A cyberpunk street at night"

## Results
The generated images successfully capture the meaning and context of the provided prompts.
